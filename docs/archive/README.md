## Archive Policy

Documents are archived when:
1. They are superseded by newer documentation
2. Content is consolidated into broader documents
3. Feature specifications are fully implemented and documented elsewhere
4. Duplication exceeds 80% with active documentation

## Restoration

To restore archived documentation:
1. Review the consolidated/replacement document
2. If content is missing, extract and update the active document
3. Do not restore without consolidating to prevent duplication
