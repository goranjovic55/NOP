# Global Project Memory

## Project Overview
NOP (Network Operations Platform) is a dashboard for network asset management, scanning, and traffic analysis.

## Architecture
- **Frontend**: React (TypeScript)
- **Backend**: Python (FastAPI/Django?)
- **Database**: PostgreSQL
- **Cache**: Redis
- **Remote Access**: Apache Guacamole (guacd)
- **Containerization**: Docker & Docker Compose

## Key Features
- Asset Discovery & Management
- Network Traffic Analysis
- Remote Access (RDP, VNC, SSH)
- File Transfer (FTP)
